# ArcFi Swap

A decentralized trading protocol built natively on Arc blockchain, using USDC as the base settlement layer.

##  What is ArcFi Swap?
ArcFi Swap is a DeFi trading protocol for the Arc ecosystem. It enables capital-efficient token swaps with minimal slippage, powered by USDC.

## Key Features
-  Fast swaps using Arc's sub-second finality
-  USDC as base trading pair
-  Secure smart contracts
-  Built on EVM-compatible Arc blockchain
